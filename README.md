# OzonPriceBot

Telegram-бот для отслеживания цен на товары с Ozon.
Бот автоматически проверяет стоимость товаров и отправляет уведомление, если цена снизилась на указанный процент.

---

## Возможности

* Отслеживание цен на товары Ozon
* Добавление нескольких товаров
* Уведомления о снижении цены
* Удаление товаров из отслеживания
* Автоматическая проверка цен по расписанию
* Хранение данных пользователей и товаров в SQLite
* Telegram-интерфейс для управления товарами

---

## Как работает бот

1. Пользователь отправляет ссылку на товар Ozon
2. Бот получает:

   * название товара
   * текущую цену
3. Пользователь указывает процент снижения цены
4. Бот сохраняет товар в базу данных
5. По расписанию запускается проверка цен
6. Если цена упала ниже заданного порога — бот отправляет уведомление

## Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/mary2517/ozonpricebot.git
cd ozonpricebot
```

---

### 2. Создание виртуального окружения

```bash
python -m venv venv
```

Активировать окружение:

#### macOS / Linux

```bash
source venv/bin/activate
```

#### Windows

```bash
venv\Scripts\activate
```

---

### 3. Установка зависимостей

```bash
pip install aiogram playwright apscheduler
```

---

### 4. Установка браузера Playwright

```bash
playwright install
```

---

### 5. Настройка токена

В файле `config.py` укажите токен Telegram-бота:

```python
BOT_TOKEN = "YOUR_TOKEN"
```

## Проверка цен

Проверка стоимости товаров выполняется автоматически два раза в день:

```python
CHECK_HOURS = [9, 21]
```

При необходимости расписание можно изменить в `config.py`.

## Особенности работы в России

Проект использует одновременно:

* Telegram API
* парсинг сайта Ozon

Из-за сетевых ограничений в России может возникнуть ситуация:

| Сервис   | Без VPN    | С VPN                      |
| -------- | ---------- | -------------------------- |
| Telegram | недоступен | доступен                   |
| Ozon     | доступен   | может работать нестабильно |

Поэтому при использовании VPN Telegram становится доступен, однако Ozon может ограничивать загрузку страниц или выдавать ошибки парсинга.

### Возможные решения

* использование split tunneling
* запуск бота на зарубежном сервере
* использование прокси отдельно для Telegram или Playwright
* запуск проекта через VPS
